# Zomato Restaurant Analysis Dashboard | Business Intelligence Project

![Zomato Power BI Dashboard](screenshots/Zomato%20Power%20BI%20Dashboard%20pic.png)

## 🎯 Project Objective

This **End-to-End Business Intelligence Project** analyzes Zomato restaurant data across 15 countries and 141 cities. The goal is to uncover key insights on market presence, customer ratings, pricing strategies, online delivery & table booking adoption, and growth trends (2010–2018) to support data-driven business decisions.

---

## 🛠️ Tech Stack

- **Microsoft Excel** – Data Cleaning, Pivot Tables & Charts
- **Power BI** – Interactive Dashboard with Slicers & KPIs
- **Tableau** – Advanced Visualizations & Geographic Mapping
- **SQL** – Data Extraction, Date Intelligence & Analytical Queries

---

## 📊 Key KPIs

- **Total Restaurants**: 9,551
- **Total Countries**: 15
- **Total Cities**: 141
- **Total Cuisines**: 1,825
- **Average Rating**: ~2.90
- **Dominant Market**: India (8,652 restaurants)

## 📁 Project Structure

- **Dashboards/** → Excel, Power BI & Tableau files
- **SQL/** → Analytical SQL queries
- **screenshots/** → Dashboard images

---

## 📸 Dashboard Views

### 1. Excel Dashboard
Static overview with Total Restaurants, Country-wise distribution, Price Bucket analysis, Month-wise & Year-wise trends.

### 2. Power BI Dashboard
Interactive dashboard with KPI cards, Rating distribution, Top Cuisines, Table Booking %, and filters.

### 3. Tableau Dashboard
Advanced visuals including Country Map, City-wise ranking, Rating buckets, and Yearly opening trends.

---

## 💾 SQL Queries Included

- Date Intelligence (Year, Month, Quarter, Financial Month/Quarter, Weekday)
- Restaurant count by Country & City
- Price Bucket Analysis
- Table Booking & Online Delivery Percentage
- Growth Trend Analysis by Year/Quarter/Month
